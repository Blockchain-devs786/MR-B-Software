# Mr B - Fast Food Order Management System

A desktop application for managing fast food orders, including Dine-in, Takeaway, and Delivery.

## Features

- **Order Management**: Create and manage orders with ease.
- **Table Management**: Track table availability for dine-in.
- **Delivery & Riders**: Assign riders and track delivery orders.
- **Expenses**: Record and categorize daily expenses.
- **Refunds**: Process refunds with reasons.
- **Reports**: Generate sales reports.
- **Receipt Printing**: Thermal receipt printing support (80mm).

## Tech Stack

- **Electron**: Desktop application framework.
- **React**: UI library.
- **TypeScript**: Type safety.
- **Tailwind CSS**: Styling.
- **MySQL**: Database.

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```
